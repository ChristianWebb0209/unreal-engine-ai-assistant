Run a material tuning pass on an existing material instance in the project.

Hard requirements:
- Find candidate material instances with asset search/registry query.
- Pick one existing material instance and adjust:
  - one scalar parameter
  - one vector parameter
- Query usage/referencers after the change.
- Save and report whether changes persisted.

Completion criteria:
- One real material instance path is modified.
- Final output lists parameter names/values changed and at least one referencer result (or explicit none).
