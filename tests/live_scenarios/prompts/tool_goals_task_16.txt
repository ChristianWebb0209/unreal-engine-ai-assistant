Run a non-destructive editor situational-awareness pass and produce a scene report.

Hard requirements:
- Use read tools first: editor snapshot, selection, scene fuzzy search, and asset index search.
- Include at least one viewport camera move and one viewport capture screenshot if those tools are available.
- Do not create/delete/rename assets in this task.
- Summarize:
  1) current selected actors
  2) top 10 actors by relevance to "gameplay"
  3) currently open editor assets (if available)
  4) screenshot path(s) created.

Completion criteria:
- Final report includes concrete actor/asset paths found via tools.
- If a tool is unavailable, explicitly mark that section "not available" and continue.
