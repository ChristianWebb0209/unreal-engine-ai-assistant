Using OpenRouter as your AI agent

Copy Markdown Open

OpenRouter provides access to multiple AI models through a single API - great for quick setup or if you want to try different models.

Setup

Create an account at openrouter.ai

Generate an API key from the dashboard

In Unreal, go to Project Settings > Plugins > Agent Integration Kit

Paste your API key in the OpenRouter API Key field
